Software estadístico

¿Qué es?
Es referido como un programa y aplicación en el que se puede realizar análisis estadísticos
Programa estadístico
Paquete estadístico ampliamente y utilizado , sirve como una herramienta para comprender los conceptos y cálculos estadísticos para la verificación de calculo realizado a mano para resolver sus problemas de tarea
Teniendo a la mano funciones estadísticas útiles para cálculos estadísticos permitiendo ver el conjunto de datos
SPSS
Software reconocido entre usuarios de Windows ,usado para la elaboración de capturas y análisis de datos para la elaboración de capturas y análisis de datos , elaboración de graficas con una data compleja.
XLSTAT
Complemento de análisis de datos de Excel potente pero flexible que permite a mas de 150 000 usuarios en mas de 120 países ,analizar ,personalizar y compartir resultados en Microsoft
